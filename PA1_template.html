My strategy:

Fill in the missing values in the dataset with the month averages.

First, calculate the average number of steps taken per month.

Then, create a new dataset (equal to the original one).

Finally, populate the variable 'steps' in the new dataset with the month averages.

# average number of steps taken per month

# create a new dataset
